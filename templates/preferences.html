Character Preference Copier

Select a source character, choose which preference files to copy, and apply them to one or more target characters.

This should point to the folder that contains account subfolders (e.g. Prefs\\AccountName).

Source Character

Characters are grouped by account. Only characters stored in your launcher configuration appear here.

Preference Items

Recommended items are pre-selected. Hover any entry to learn what it contains.

Destination Characters

Check every character that should receive the copied preferences. Source character is disabled.
